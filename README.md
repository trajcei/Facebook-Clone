# FacebookClone

A clone of the Facebook app made with React.

## Tech used:

* React + React HOOKS

* React Context API -> Redux Pattern

* FLEXBOX

* Material UI

* Firebase's Firestore realtime DB

* Firebase Hosting

* Firebase Google Authentication
